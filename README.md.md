версия python 3.13

сначала появляется и обрабатывается датасет из файла rotten_tomatoes.csv. файл rotten_tomatoes.csv был найден на [Rotten Tomatoes Top Movies Ratings and Technical](https://www.kaggle.com/datasets/thedevastator/rotten-tomatoes-top-movies-ratings-and-technical)

затем обучаются модели knn. эта модель была выбрана логически, поскольку она лучше подходит под задачу.
модели две, первая предсказывает рейтинг критиков, вторая - зрителей.
затем проверяем, какое число соседей даст нам наибольшую точность, и делаем предсказания. предсказания записываются обратно в датасет.

дальше сайт. сайт имеет две страницы, на главной есть список фильмов на выбор. на странице @app.route('/find') выводится результат для фильма: название, год, режиссер, предсказания обеих моделей и обе настоящие оценки.
